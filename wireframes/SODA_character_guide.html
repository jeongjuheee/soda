SODA 마스코트 — 캐릭터 디자인 가이드

개발자(안티) 전달용. 캐릭터 형태, 레벨별 외형, 표정 시스템, 모션 스펙을 정리한 디자인 가이드입니다.
이 문서를 기반으로 캐릭터를 일러스트로 제작하거나 SVG/코드로 구현하면 됩니다.

캐릭터명: 소다 형태: 물방울 블롭 메인컬러: #4A7BF0 레벨: 4단계 표정: 5종
1기본 형태 (Base Shape)

실루엣

  • 물방울(teardrop) 형태가 기본. 위는 둥글고 아래로 갈수록 약간 좁아지는 통통한 비정형 블롭
  • 완전히 대칭적이지 않아도 됨 — 살짝 비정형이어야 "말랑한 생물체" 느낌이 살아남
  • 하단에 살짝 꼬리(또는 끝이 좁아지는 형태)가 있으면 물방울 정체성 강화

외곽선 (Outline)

  • 화이트 보더(외곽선) 필수 — 어떤 배경에서도 캐릭터가 또렷하게 분리되어 보이게 하는 핵심 디테일
  • 굵기: 3~4px (캐릭터 크기에 따라 비례 조정)

색상 (Body Color)

기본 블루 그라데이션 — 좌상단 밝게, 우하단 진하게

#6F8FE8 — 졸림/슬픔 상태에서 채도 낮은 톤으로 전환

Lv.4 빛나는소다 — 오로라 느낌의 다색 그라데이션으로 진화

하이라이트 (Highlight)

  • 좌상단에 큰 타원 하이라이트 1개 (opacity 0.4~0.55, fill white)
  • 바로 아래 또는 좌측에 작은 타원 하이라이트 1개 (opacity 0.6~0.75) — 글로시/펄 머티리얼 느낌
  • 우하단에 작은 림 라이트(rim light) 타원 1개 (opacity 0.15~0.2)

눈 (Eyes)

  • 기본: 세로로 약간 긴 타원 2개, fill #16234F (다크 네이비)
  • 눈 안에 흰색 하이라이트 점 1개씩 — 생동감 핵심 디테일
  • 두 눈 사이 간격: 몸통 너비의 약 25~30%

그림자

  • 발 아래(캐릭터 하단)에 블루 계열 타원 그림자 (fill #4A7BF0, opacity 0.10~0.15, blur 처리)
  • 부유 모션 시 캐릭터가 올라갈 때 그림자 작아지고, 내려올 때 커짐 (공중 부유감 강조)
레퍼런스 분위기: 레퍼런스로 확정한 무드는 "귀여우면서 약간 멍청한(?) 느낌, 형태가 고정되지 않고 말랑말랑하게 움직이는" 캐릭터. 너무 정교하게 다듬은 것보다 약간 뭉툭하고 불완전한 느낌이 있어야 함.

4모션 스펙 (Motion Spec)

① 대기 모션 (Idle / Floating)

캐릭터가 화면에 정적으로 있을 때 항상 실행되는 기본 모션.

  • 동작: 위아래 부유 (translateY: 0 → -6px → 0)
  • 주기: 3.2초 ease-in-out, 무한 반복
  • 회전: 살짝 좌우 기울기 추가 가능 (rotate -1.5° → 0 → 1.5°)
  • 그림자: 캐릭터 상승 시 그림자 작아짐 (scale 0.75), 하강 시 커짐 (scale 1.0) — 공중 부유감 강조
  • 눈 깜빡임: 4.5초마다 1회, scaleY 1.0 → 0.1 → 1.0 (0.1초 빠르게)
  • 홈/챌린지 탭에 배치된 캐릭터들은 서로 다른 위상(phase)으로 부유 — 일렬로 똑같이 움직이면 기계적으로 보임

② 탭 반응 모션 (Squash & Stretch)

유저가 캐릭터를 탭할 때 발생하는 인터랙션 모션.

키프레임transform타이밍
0%scale(1, 1)시작
30%scale(1.18, 0.78)눌림 (납작해짐)
55%scale(0.88, 1.16)튀어오름 (늘어남)
75%scale(1.05, 0.95)착지 (살짝 눌림)
100%scale(1, 1)복귀
  • duration: 0.55초
  • easing: cubic-bezier(0.36, 1.6, 0.4, 1) — 탄성 있는 오버슈트 느낌
  • transform-origin: center bottom (발 쪽이 고정된 채로 움직이는 느낌)

③ 레벨업 모먼트 모션

SP가 레벨 임계치를 초과하는 순간 1회 트리거.

  • 루틴 완료 축하 표정으로 전환
  • 스쿼시앤스트레치를 더 크게 (scale 범위 1.25 수준으로 확대)
  • 주변 별 모양 파티클 5~7개 방사형으로 퍼짐 (opacity 1.0 → 0, translateX/Y 방사)
  • 새 액세서리 페이드인 (opacity 0 → 1, 0.4초)
  • 레벨명 텍스트 팝업 (예: "반짝소다로 진화!") 잠깐 노출 후 사라짐

④ 응원 흔들림 모션 (Wobble)

응원·독려 표정일 때 추가로 실행.

  • 좌우 흔들기: rotate -4° → 0 → +4° → 0, 2초 ease-in-out 반복
  • idle 부유 모션과 동시 실행

⑤ 처짐 모션 (Droopy)

졸림 표정일 때 대기 모션 대체.

  • translateY: 0 → +3px → 0, scale: 1 → 1.02, 0.98 (살짝 퍼짐)
  • 4초 ease-in-out 반복 — 부유보다 느리고 무거운 느낌
