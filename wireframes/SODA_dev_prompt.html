모션 가이드

  • 대기(idle): 위아래 부유 (translateY 0 → -6px → 0), 3.2초 ease-in-out 무한 반복. 발 아래 그림자 타원이 함께 커졌다 작아지며 공중 부유감 강조
  • 탭 반응: 스쿼시앤스트레치 — scale(1,1) → scale(1.18, 0.78) → scale(0.88, 1.16) → scale(1,1), 0.55초 cubic-bezier(.36,1.6,.4,1)
  • 레벨업 모먼트: 루틴 완료 축하 모션 확대 버전 + 반짝이는 파티클(별 모양) 주변 퍼짐
3SP(소다포인트) 성장 시스템

포인트 획득 방식

행동획득 SP
루틴 1개 인증+10 SP
챌린지 완주챌린지별 상이 (30~80 SP)
7일 연속 스트릭 달성+30 SP
중요: 외부 SNS 공유, 친구 초대 등 외부 행동은 포인트 대상 아님. 순수 개인 루틴·챌린지 활동으로만 적립.

SP 표시 위치

  • 홈 화면 히어로 카드 내 진행 바 (현재 SP / 다음 레벨 목표 SP)
  • 챌린지 탭 상단 미니 위젯 (축약 표시)
  • 챌린지 카드 / 홈 루틴 카드 — 획득 예정 SP 표시 (+10 SP 등)
  • 마이페이지 — 전체 누적 SP 및 레벨 히스토리
4탭별 화면 스펙
TAB 1 — 홈

구성 요소 (위 → 아래)

  1. 상단 인사말 + 날짜: "오늘도 화이팅, {이름}님" / 날짜, 우측 알림 벨 버튼
  2. 히어로 카드 (메인컬러 그라데이션 배경):
    • 좌상단: 레벨명 + 레벨 번호 칩 (예: "반짝소다 · Lv.3")
    • 중앙: 소다 캐릭터 (현재 레벨 액세서리 + 현재 표정 로직 적용) — 탭하면 스쿼시 모션
    • 캐릭터 하단: 말풍선 (상황별 동적 텍스트)
    • 하단: SP 진행 바 + "XXX / XXXX SP · 다음 진화까지 XXX SP"
  3. 탭 힌트 텍스트: "👆 소다를 탭해서 쓰다듬어 주세요" (작은 그레이 텍스트)
  4. 오늘의 루틴 카드:
    • 상단: "오늘의 루틴" 제목 + "X / X 완료" 카운트
    • 진행 바 (그라데이션)
    • 루틴 아이템 리스트: 아이콘 + 루틴명 + 획득 SP / 완료 여부. 미완료는 우측 "인증" 버튼(메인컬러), 완료는 "완료 ✓" 텍스트
  5. 오늘의 인사이트 티저 카드: 별 아이콘 + "오늘의 인사이트" 라벨 + 아티클 제목. 탭 시 인사이트 탭으로 이동
TAB 2 — 챌린지

구성 요소

  1. 미니 소다 위젯: 캐릭터 축소 버전 + 레벨명 + SP + 진행 바 + 우측 화살표(→ 탭 시 캐릭터 상세 화면 이동)
  2. 필터 필 (진행중 / 완료 / 전체): 활성 탭만 메인컬러
  3. 참여중인 챌린지 리스트: 카드 클릭 시 상세 바텀시트 오픈
    • 카드 구성: 카테고리 태그 + 챌린지명 + D-day + 참여인원 / 원형 진행률 / 완주 시 SP
  4. 이런 챌린지는 어때요? 섹션: "전체보기 →" 우측 배치
    • 카드 구성: 추천 유형 태그 + 챌린지명 + 카테고리 필 + 참여인원 필 + "참여" 버튼(메인컬러)
    • "참여" 클릭 → 버튼 "참여중"으로 변경(비활성) + 참여중인 챌린지 리스트 맨 위로 추가
    • 참여인원 0이면 참여중인 챌린지 없음 상태 → 빈 상태(empty state) 노출

챌린지 상세 (바텀시트)

참여중인 챌린지 카드 탭 → 하단에서 슬라이드업되는 모달. 닫기: ✕ 버튼 / 배경 탭 / 아래로 드래그

  • 카테고리 태그 + 챌린지명 + D-day · 참여인원
  • 원형 도넛 그래프 (달성률 %) — conic-gradient 또는 SVG stroke-dasharray
  • 정보 3분할: 진행 일수 / 완주 보상 SP / 연속 인증(🔥 N일) — 하루라도 인증 건너뛰면 streak 0 리셋
  • 최근 인증 기록 (최근 10일): 완료(블루 체크) / 오늘(블루 테두리 숫자) / 예정(그레이 숫자)
  • 하단 고정 버튼: "오늘 루틴 인증하기" (메인컬러) + "챌린지 그만하기" (연한 빨강)
    • 이미 오늘 인증했으면 버튼 비활성 + "오늘 인증 완료 ✓"로 텍스트 변경
    • "그만하기" 탭 → 확인 모달 후 리스트에서 제거
TAB 3 — 인사이트

구성 요소
