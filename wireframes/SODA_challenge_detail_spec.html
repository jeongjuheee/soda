상태표시 조건
완료 (done)해당 일차에 인증 기록이 있는 경우 — 블루 배경 + 체크 아이콘
오늘 (today)현재 진행중인 일차, 아직 인증 전 — 화이트 배경 + 블루 테두리 + 일차 숫자

최근 10일 정도만 노출, 전체 기록은 별도 "전체 기록 보기"로 확장 가능 (현재 와이어프레임엔 미포함, 필요 시 추가 논의)
6하단 액션 버튼
버튼동작
오늘 루틴 인증하기탭 시 인증 처리 → doneDays +1, streak +1, SP 지급(+10 SP 등 루틴 단위 보상) → 완료 토스트 노출 → 시트 닫힘
챌린지 그만하기탭 시 확인 모달("정말 그만두시겠어요?") → 확인 시 참여중 리스트에서 제거, 진행 데이터 삭제
"오늘 루틴 인증하기" 버튼은 이미 오늘 인증을 완료한 상태라면 비활성화 처리(회색) + 텍스트를 "오늘 인증 완료 ✓"로 변경 필요
