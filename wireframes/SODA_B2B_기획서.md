# SODA 기업용(B2B) 기획 설계 정리 (Claude 점검 완료본)

---

## 1. 기업 유저 타겟
단순 스펙보다 조직문화 핏이 맞아 조기 퇴사하지 않을 주니어 인재를 찾는 스타트업 및 중소·중견기업 채용담당자.

---

## 2. 수익 모델 (확정)

```
무료    기업 온보딩 + 매칭률 상위 3명 미리보기 (익명, 성향 유형 + 매칭률만)
구독    전체 인재풀 열람 + 루틴 지속성 데이터 + 상세 리포트
        월 9.9만 ~ 19.9만원 (스탠다드 / 프리미엄)
수수료  채용 완료 시 연봉의 5~7%
```

경쟁사 비교:
- 원티드: 채용 완료 시 연봉 7% 수수료
- SODA: 구독 + 채용 수수료 5~7% (진입 장벽 낮게, 수수료율 낮게)

---

## 3. 기업 온보딩 (확정)

- 사업자등록번호 인증 → 담당자 정보 입력
- 15문항 조직문화 진단 한 번에 완료
- UI 포지셔닝: "우리 조직의 비즈니스 DNA를 분석합니다 (약 3분 소요)"
- 진행률 바 표시 + 중간 저장 가능
- 완료 시 기업 성향 유형 결과 카드 바로 노출

---

## 4. 스카웃 제안 3단계 (확정)

```
구직중 OFF      기업에 노출 안 됨, 제안 불가
관심있어요      노출은 되지만 알림 없이 마이페이지에 조용히 쌓임
구직중 ON       제안 알림 수신 + 수락/나중에 액션 가능
```

수락 시: 유저 연락처 공개 → 기업이 직접 연락
채용 완료 시: 수수료 발생

---

## 5. 기업용 5개 탭 UI 확정

### ① 홈 탭 (A안 확정)
```
상단  회사명 + 오늘 신규 매칭 인원 안내
중단  매칭/제안/수락 지표 카드 3개
      오늘 신규 매칭 인재 리스트 (익명 + 성향 + 매칭률)
하단  스카웃 응답 알림 카드 (수락 시 연락처 확인 유도)
```

### ② 인재풀 탭 (A안 확정)
```
상단  헤더 + 매칭 인재 수
      필터 탭: 전체 / 구직중 / 관심있어요
      정렬: 매칭률순 / 루틴달성순 / 최신순
중단  인재 카드 리스트
      - 익명 + 성향 유형 + 지망 직무
      - 루틴 달성일 + 연속 달성 + 구직 상태
      - 스카웃 제안하기 버튼 (구직중 OFF면 비활성)
하단  구독 유도 배너 (미구독 시)
```

### ③ 제안함 탭 (A안 확정)
```
상단  헤더 + 총 발송 건수
      탭: 진행중 N / 수락 N / 거절 N
중단  제안 카드 리스트
      - 익명 + 성향 + 매칭률
      - 제안 날짜 + 상태 뱃지 (수락/대기중/거절)
      - 수락 시: 연락처 공개 안내 카드 강조
```

### ④ 인사이트 탭 (A안 확정 — 개인용과 동일 구조, B2B 콘텐츠)
```
상단  헤더 + 검색 아이콘
      이번 주 추천 가로 스크롤 (3~4장)
      카테고리 필터: 전체 / HR트렌드 / 채용전략 / 조직문화
하단  아티클 피드 (제목 + 키워드 태그 + 북마크)
```

콘텐츠 방향 (개인용과 다름):
- HR트렌드, 채용전략, 조직문화, 온보딩, 리텐션, 주니어 성장 관리

### ⑤ 마이페이지 (A안 확정)
```
상단  헤더 + 설정 아이콘
      기업 프로필 카드 (로고 + 회사명 + 성향 유형 + 구독 상태 + 갱신일)
중단  지표 카드 3개 (매칭 인재 / 수락 / 발송)
      기업 DNA 바 차트 (5축 전체)
      매칭 데이터 리포트 (우리 매칭률 vs 업계 평균)
하단  구독 관리
      저장한 인사이트
```

## 6. 인재 카드에 노출되는 데이터

```
익명 처리      이름/사진 비공개, 성향 유형 + 지망 직무만 표시
매칭률         SODA 5축 매칭 알고리즘 계산값 (%)
루틴 지속성    총 달성일 + 연속 달성일 (행동 데이터)
구직 상태      구직중 / 관심있어요 / OFF
```

수락 후 공개:
- 유저 연락처 (이메일 or 전화번호)

---

## 7. 데이터 고도화 로드맵

```
초기  15문항 정적 데이터로 기업-개인 매칭
중기  루틴 인증 행동 데이터 누적으로 매칭 정확도 보정
장기  실제 채용 후 재직 기간 데이터로 매칭 알고리즘 검증
```

서비스 포지셔닝:
"스펙만 보고 뽑았다가 3개월 만에 나간 주니어, 문제는 스킬이 아니라 비즈니스 핏이었습니다."

---

## 8. 기업용 하단 탭 구성

| 탭 | 아이콘 | 역할 |
|---|---|---|
| 홈 | 🏠 | 오늘의 매칭 현황 + 스카웃 알림 |
| 인재풀 | 👥 | 매칭 인재 열람 + 제안 |
| 제안함 | 📬 | 발송한 제안 현황 관리 |
| 인사이트 | 💡 | HR 트렌드 + 채용 전략 아티클 |
| 마이페이지 | 🏢 | 기업 DNA + 리포트 + 구독 관리 |
